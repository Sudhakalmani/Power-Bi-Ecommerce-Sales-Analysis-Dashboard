# E-commerce Sales Dashboard – Power BI
## Project Overview
This project presents an interactive Power BI dashboard designed to analyze E-commerce sales performance.
The dashboard provides insights into sales trends, product performance, regional sales distribution, and profit analysis to support data-driven decision making.

## Key Metrics
• YTD Sales: $11.53M  
• YTD Profit: $1.34M  
• Profit Margin: 11.58%  
• Quantity Sold: 107.2K  

## Dashboard Features
- Sales by Category
- Top 5 Products by Sales
- Bottom 5 Products by Sales
- Regional Sales Analysis
- Shipping Type Analysis
- State-wise Sales Visualization

## Tools Used
- Power BI
- Data Visualization
- Data Modeling
- DAX

## Key Insights
- Office Supplies generated the highest revenue.
- Profit increased by 4.5% compared to the previous year.
- Standard shipping contributes the largest portion of sales.
- Some products show lower sales performance which can be improved with marketing strategies.
